Section 1
Overview

The Affiliate Payout Calculator is a web-based tool designed to help the tattoo studio quickly and accurately calculate commission earnings for introducers across three tiers — L1, L2, and L3. It mirrors the logic previously managed in a spreadsheet, but works directly in any web browser with no installation required.

Results update automatically as you type — just like a live spreadsheet — so there is no need to click a "calculate" button. Once you are happy with a scenario, you can save it to a running log of client records at the bottom of the page.

KEY BENEFITS

Live calculation — Results appear instantly as inputs change — no manual refresh needed.

Browser-based — Works on any device with a modern browser. No software to install.

Dark & Light mode — Automatically adapts to your system colour preference.

Client log — Save and review multiple client scenarios in a single session.

Accurate logic — All payout rules are encoded directly from the original spreadsheet.

Section 2
Understanding the Tier System

The studio operates a three-level affiliate introducer programme. When a client books through an introducer, that person earns a commission based on their tier level. Higher tiers (L2, L3) also trigger smaller secondary commissions for introducers above them in the chain.

TierRoleEarns OnAlso Triggers
L1Direct introducerTheir own referrals onlyNothing
L2Mid-tier introducerTheir own referralsL1 earns a secondary cut
L3Senior introducerTheir own referralsBoth L1 and L2 earn secondary cuts

Note: Secondary commissions are only triggered on Full Sign-Up bookings, not Deposit Only.

Section 4
How to Use the Calculator

Follow these steps each time you need to calculate a payout:

Step 1 — Open the file
Open this HTML file in any modern web browser (Chrome, Safari, Firefox, Edge).
Step 2 — Enter the client name
Type the client's name in the Client Name field. This is for record-keeping only and does not affect any calculation.
Step 3 — Select the service type
Choose either Signature or Premium. Signature services have higher base commission rates than Premium.
Step 4 — Select the booking status
Choose Full Sign-Up if the client has paid in full, or Deposit Only if they have only paid a holding deposit. This is the single most impactful field — deposit-only bookings pay flat commissions, while full sign-ups pay base amounts plus a percentage of the tattoo price.
Step 5 — Select the introducer tier
Choose L1, L2, or L3. This determines which result cards show non-zero values and whether secondary commissions are triggered.
Step 6 — Enter the tattoo price
Type the agreed tattoo price in dollars. For Full Sign-Up bookings, a percentage of this amount is added to the base commission. For Deposit Only, this field has no effect.
Step 7 — Read the results
The five coloured result cards update instantly. Review each tier's earnings, the total payout, and the company net revenue.
Step 8 — Save if needed
Click + Save Current to add this client to the session log. Multiple entries for the same client name are automatically grouped into a single summary row showing combined totals — click the row to expand or collapse the individual entries.
TIP: You do not need to clear the form between clients. Simply update any field and the results refresh immediately.
Section 5
Understanding the Results
5.1   When a card shows $0

Not all five cards will show a non-zero value for every booking. This is expected behaviour:

If the introducer is L1, only the L1 card will show earnings. L2 and L3 will be $0.

If the introducer is L2, L2 shows full earnings. L1 shows a secondary cut (Full Sign-Up only). L3 is $0.

If the introducer is L3, L3 shows full earnings. Both L1 and L2 show secondary cuts (Full Sign-Up only).

5.2   Deposit Only vs Full Sign-Up

The Status field has the greatest impact on payout amounts:

Deposit OnlyFull Sign-Up
Signature — direct tier$200 flat$1,000 + 5% of tattoo price
Premium — direct tier$100 flat$400 + 5% of tattoo price
Secondary tiers (L1/L2 upline)$0 — not triggered$300 + 2% (Sig.) / $150 (Prem.)
5.3   Company Net Revenue

Calculated as gross studio revenue minus all affiliate commissions. Gross revenue by service and status:

Signature Full Sign-Up: $6,000 + tattoo price

Premium Full Sign-Up: $2,000 + tattoo price

Signature Deposit Only: $500 flat

Premium Deposit Only: $300 flat

Section 6
Saving & Managing Client Records

The Saved Clients table lets you build a log of multiple clients within a single session. Entries for the same client name are automatically grouped, with totals shown at the group level.

How grouping works

When you save more than one entry under the same client name, the table consolidates them into a single grouped row. The grouped row displays the sum of all individual values — Price, L1, L2, L3, Total Payout, and Net Revenue — giving you an instant combined total for that client. A teal number badge shows how many entries are in the group.

ActionHow
Save a recordClick + Save Current — the entry is added and automatically grouped if the client name already exists.
Expand a groupClick the grouped row to reveal all individual entries underneath. The arrow rotates to show the group is open.
Collapse a groupClick the grouped row again to hide the individual entries and return to the summary view.
Remove one entryExpand the group, then click the Remove button on the individual entry you want to delete. If only one entry remains, the group automatically becomes a plain single row.
Permanent storageData is not saved between sessions. Screenshot or copy values before closing the browser.
REMINDER: Always record your saved clients before closing the browser tab.

Section 8
Frequently Asked Questions
Q: Can I use the calculator on a phone or tablet?
A: Yes. The layout adapts to smaller screens. For the best experience on mobile, view it in landscape orientation.
Q: Why are L2 and L3 showing $0 when the introducer is L1?
A: This is correct. An L1 introducer is the only person in the chain for that booking. No secondary commissions are triggered because there are no higher-tier introducers above them.
Q: Why does changing the tattoo price have no effect on Deposit Only bookings?
A: Deposit Only bookings always pay a fixed flat fee regardless of the tattoo price. The percentage-based addition only applies to Full Sign-Up bookings where the client has paid in full.
Q: Why is the L1 card showing a value even though the introducer is L3?
A: When the direct introducer is L3, both L1 and L2 in the chain receive a secondary upline commission — but only on Full Sign-Up bookings. If the status is Deposit Only, those secondary cards will correctly show $0.
Q: What is the difference between Signature and Premium payouts?
A: Signature services carry higher base commissions. On a Deposit Only booking, Signature pays $200 versus $100 for Premium. On a Full Sign-Up, Signature pays $1,000 base plus 5%, while Premium pays $400 base plus 5%. Secondary upline commissions also differ: Signature pays $300 + 2% while Premium pays a flat $150.
Q: Can a booking ever result in all five cards showing non-zero values?
A: Yes — but only when the introducer is L3 and the booking is a Full Sign-Up. In that scenario L3 earns the direct commission, both L2 and L1 earn upline secondary commissions, the Total Payout shows the combined sum, and Company Net Revenue shows what remains.
Q: If I save multiple scenarios for the same client, how does the table handle them?
A: The table automatically groups all entries under the same client name into a single row, showing the combined totals for that client. Click the grouped row to expand it and see each individual entry — each with its own Remove button. This is useful when a client has multiple bookings or when you want to compare different scenarios side by side before committing to a record.
